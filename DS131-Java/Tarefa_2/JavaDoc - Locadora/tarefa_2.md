## Exercicio 4:

do segundo grau do teclado e mostre os valores de x da equação;

$x = (-b+-sqrt(b^2-4ac))/2a$

a. Verifique na documentação Java as operações matemáticas da classe
Math;
b. Não é necessário validar tipos de raízes;
c. Dados para teste:
- x^2 -9 = 0 Resultado: x = 3 e x = -3
- x^2 -9x = 0 Resultado: x = 0 e x = 9
- 3x^2 -7x + 2 = 0 Resultado: x = 1/3 e x = 2
- -x^2 + 4x – 4 = 0 Resultado: x = 2 e x = 2
